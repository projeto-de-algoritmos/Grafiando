## Sobre 
O objetivo dessa aplicação é demonstrar quantos pontos são necessários para gerar um labirinto, sempre usando uma aresta até o próximo ponto.  Vamos utilizar a busca em profundidade. O desafio foi baseado na questão [1076](https://www.urionlinejudge.com.br/judge/pt/problems/view/1076) da plataforma URI-Online Judge.
